# Dark-Scanner 개인정보처리방침

**최종 수정일:** 2026-03-16

---

## 요약

**Dark-Scanner는 어떠한 개인정보도 외부 서버로 전송하지 않습니다.**
방문한 웹페이지의 내용, DOM 데이터, 네트워크 응답, 분석 결과 등 모든 처리는
사용자의 브라우저 내부에서만 수행됩니다.

---

## 1. 수집하는 정보

Dark-Scanner는 다음 정보를 **브라우저 내부에서만** 처리합니다.

| 정보 유형 | 처리 목적 | 외부 전송 |
|-----------|-----------|-----------|
| 방문 페이지의 DOM 구조 | 다크 패턴 요소 탐지 | **없음** |
| 페이지 텍스트 (상품명, 리뷰, CTA 문구 등) | NLP 키워드 분석 | **없음** |
| fetch/XHR 응답 데이터 | 실시간 수치 서버 검증 | **없음** |
| 탐지 결과 (DarkPatternDetection) | 팝업 및 오버레이 표시 | **없음** |

---

## 2. 저장 방식

| 저장소 | 저장 내용 | 유지 기간 |
|--------|-----------|-----------|
| `chrome.storage.session` | 현재 탭의 탐지 결과 | 탭 닫힘 시 자동 삭제 |
| `chrome.storage.local` | FOMO 키워드 사전 캐시 | 확장 프로그램 삭제 시 제거 |

**브라우저 이외의 어떤 서버나 데이터베이스에도 정보를 저장하지 않습니다.**

---

## 3. 사용하는 권한 및 목적

| 권한 | 사용 목적 |
|------|-----------|
| `webRequest` | 현재 탭의 네트워크 응답 분석 (읽기 전용, 차단 불가) |
| `storage` | 탐지 결과 및 키워드 사전 로컬 저장 |
| `scripting` | Content Script 주입 (DOM 스캔) |
| `offscreen` | WASM 실행 환경 확보 (NLP 모델 예정) |
| `tabs` | 현재 활성 탭 식별 (배지 표시) |
| `<all_urls>` | 모든 웹페이지에서 다크 패턴 탐지 |

---

## 4. 제3자 공유

Dark-Scanner는 수집된 어떠한 정보도 제3자와 공유하지 않습니다.
광고 네트워크, 분석 서비스, 외부 API와 연동되지 않습니다.
